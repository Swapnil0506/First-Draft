# First-Draft

A systematic blog app named <b>First Draft</b> where you can do the following :-
<br>-> Register yourself
<br>-> Create a profile
<br>-> Update your profile 
<br>-> Create blogs (also update/delete them)
<br>-> Read others' blogs
